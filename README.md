# Huggy Gomboc

Huggy Gomboc is a small emotional companion robot prototype.

## V1 Goal

The first version will be a simple reactive system:

Planned reactions:
- touch or button input
- LED heartbeat effect
- small servo movement
- buzzer sound

## Magyar leírás

Az Ölelő Gombóc célja egy kicsi companion robot építése, amely érintésre, közelségre és később hangra reagál fénnyel, mozgással és egyszerű személyiségállapotokkal.
